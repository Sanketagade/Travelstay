# Travelstay MERN Stack Travel Accommodation Platform
A full-stack travel accommodation platform built with the MERN stack, enabling users to manage listings and reviews through a secure, responsive interface

# Project Overview
TravelStay is designed to provide users with a smooth experience for browsing and managing travel accommodations.  
The project demonstrates **end-to-end MERN stack development**, including frontend UI, backend API, database integration, and authentication workflows.

##  Key Features
- Browse travel accommodations with visually rich listing cards  
- Search destinations with a global search bar  
- Category-based browsing (Trending, Rooms, Cities, Mountains, Pools, Camping, etc.)  
- Secure user authentication system (Signup & Login)  
- Full CRUD operations for accommodation listings (Create, Read, Update, Delete)  
- Listings include title, description, location, country, price, and images  
- Review system with star ratings and comments  
- Input validation using Joi for secure and accurate data handling  
- RESTful backend API with scalable architecture  
- Fully responsive UI across devices
